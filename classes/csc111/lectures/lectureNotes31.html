The same program in multiple languages

Today we will look at the code for a simple program as written in multiple languages.


Python

for k in range(10,0,-1):
  print (k)

Try it online!


Java

class revl {
  public static void main(String [] args) {
    int k;
    for (k=10; k>=1; k--) {
      System.out.println(k);
    }
  }
}

Try it online!


C

#include <stdio.h>
int main() {
  int k;
  for (k=10; k>=1; k--) {
    printf("%d\n",k);
  }
  return 0;
}

Try it online!


C++

using namespace std;
#include <iostream>
int main() {
  int k;
  for (k=10; k>=1; k--) {
    cout << k << endl;
  }
  return 0;
}


Try it online!
