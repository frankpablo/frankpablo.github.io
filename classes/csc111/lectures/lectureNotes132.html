What is List Comprehension

List comprehension offers a shorter syntax when you want to create a new list based on the values of an existing list.

This approach requires a deeper understanding of how python works but let's you write fancier code.


It follows the mathematical notation for describing sets: \[ P = \{ x \in \mathbb{R} \mid x > 0 \} \]

or

\[ E = \{ x \in \mathbb{R} \mid x \mod 2 \text{ is equal to 0} \} \]


Example 1:



Using a for loop



squares = []
for i in range(10):
    squares.append(i * i)

Try it online!


Using map() Objects

map() needs a function and an iterable, and it will create an object (we didn't see this in class until now). 

TAX_RATE = .08
def get_price_with_tax(txn):
    return txn * (1 + TAX_RATE)
# makes the iterable "final_prices"
final_prices = map(get_price_with_tax, txns)
list(final_prices)


Try it online!


Using List Comprehensions

you could rewrite the for loop from the first example in just a single line of code: 

squares = [i * i for i in range(10)]


Try it online!

This approach follows this format: 

new_list = [expression for member in iterable]


Note the key words: for and in


Every list comprehension in Python includes three elements

:
  1. expression is the member itself, a call to a method, or any other valid expression that returns a value. In the example above, the expression i * i is the square of the member value.
  2. member is the object or value in the list or iterable. In the example above, the member value is i.
  3. iterable is a list, set, sequence, generator, or any other object that can return its elements one at a time. In the example above, the iterable is range(10).


You can think of it in reverse:

For every member in the original iterable, apply an expression and add to the "new_list"


Example 2

We'll rewrite the "Tax" Example using list comprehension too: 

txns = [1.09, 23.56, 57.84, 4.56, 6.78]
TAX_RATE = .08
def get_price_with_tax(txn):
    return txn * (1 + TAX_RATE)
final_prices = [get_price_with_tax(i) for i in txns]

Advanced Version (Conditionals)

A more complete description of the comprehension formula adds support for optional conditionals. The most common way to add conditional logic to a list comprehension is to add a conditional to the end of the expression: 

new_list = [expression for member in iterable (if conditional)]


Example 3: list comprehension with conditionals

: 
sentence = 'the rocket came back from mars'
vowels = [i for i in sentence if i in 'aeiou']
