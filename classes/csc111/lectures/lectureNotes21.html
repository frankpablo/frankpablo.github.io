The Random Module

One of the most useful modules is the random module.

It let's you:
  • generate a random range: random.randrange(stop) or random.randrange(start, stop[, step])
  • generate a random number in an interval: random.randint(a, b)
  • extract a random choice from a sequence: random.choice(seq)
  • shiffle the items in a sequence (in place): random.shuffle(x[, random]
  • and many more
Please check out this reference Documentation for Random
