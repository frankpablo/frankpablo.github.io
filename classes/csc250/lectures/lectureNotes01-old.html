One possible answer:

Computer Science is the structured study of, and the design of solutions for, problems that are solvable with an algorithm.

And what is an algorithm?
One possible answer:
(Wait; then Click)

A series of steps that allow you to solve a problem.
(there is a bit more than that, but for now, that's enough)



another possible definition: "A mathematical set of tools for understanding (just about) any system"

  • Computer Science is not simply a collection of data to be memorized.
    CS is a way to approach the analysis of a problem and the design of a solution for it.
  • The fact that it has "computer" in the name is an indication of the way we think of and design solutions for these problems.
    A computer requires structure and organization to be able to function and so, CS is the discipline that teaches one to think in a structured and organized manner of ways to analyze and solve problems.
    However, we can do CS wisthout computers as long as we think the right way.

    "Computer Science is no more about computers than astronomy is about telescopes"

    Edsger Dijkstra MIT
    Dijkstra was a Dutch Computer Scientist that was good (The Serena Williams of CS).

  • You will see many similarities with engineering, mathematics, and physics because 1) CS is the descendant those fields, and 2) it inherits the scientific method.
  • One distinction betwee CS and Physics is that Physics attempts to formulate general laws from observed patterns while CS attempts to explore the imits of computatioon given a set of basic rules.

    Sort of like seeing what can be buillt given a few types of Lego pieces.

Some processes are easy to do but hard to reverse:


So we could use this idea to easily "hide" each parties info until it is time to reveal it.

For example: multiplying two numbers together is pretty easy:
\[ 2399 * 7703 = ?\]

However, factoring a number can be quite hard:
\[25240577 = A * B\]

So, How does this help with our phone roulette?

Imagine these steps:
  1. The Casino and you agree that you will give them a number that has a factor that signifies "red" or one that signifies "black".
    Let's say that if we use a factor that ends in "7", that means "red", if we don't, that means "black".
    Example: \( 18479497 = 2399 * 7703\) means that you chose "black" (no factor ends in "7").
  2. You tell the dealer your number (18479497).
  3. The dealer spins the roulette and says the result out loud. Let's say they got "black"
  4. You tell the dealler your factors (2399, and 7703) and they verify if you got it right or wrong by multiplying.

Class Objectives

This class has three main objectives:

  • Learn the language of computation: learning how to understand, analyze, and solve problems through the language of computation (not programming).
  • Try to answer: what is doable with a computer?: Explore the limits of computation.
  • Try to answer: What is efficiently doable with a computer: Explore computational complexity.


The topics we'll explore include:
  • Logic and proofs
  • Regular languages and expressions
  • Context-free languages
  • Finite automata
  • Turing machines
  • Decidability
  • Enumerability
  • Mapping reducibility
  • Computational complexity
  • Advanced topics
  • Some fun examples from cryptography, ML, and more
