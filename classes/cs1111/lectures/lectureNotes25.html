Lecture Notes 25: Intro to Parsing.


Objectives

During this module, we'll explore the idea of parsing and the following elements:

  • markup languages that are easy to parse
  • Java structures that help with parsing
  • parsing concepts like "tags", "tokens", etc.



Before Starting

If you do not have your Codio course ready, use any text editor or simple IDE. Some possibilities are:

  • Write using Sublime; Compile and run in the Terminal
  • Use IDEs like: IntelliJ, DrJava, or JGrasp
